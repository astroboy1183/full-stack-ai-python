*********************************
PYDANTIC
*********************************
Pydantic is a Python library used for data validation, parsing, and settings management using Python type hints.

1. Data Validation

Pydantic ensures that incoming data matches the expected types and structure.

--> Automatically validates input data
--> Converts data into correct types when possible
--> Raises clear errors for invalid data

2. Data Parsing & Validation

--> Converts raw input (JSON, dict, API data) into Python objects
--> Handles type coercion (e.g., string to int)
--> Ensures clean and reliable data flow

3. API Development

--> Widely used with frameworks like FastAPI
--> Defines request/response schemas
--> Ensures data consistency between client and server

4. Configuration Management

--> Used to manage environment variables and settings
--> Helps structure application configs cleanly
--> Supports .env files and environment-based configs

5. Serialization / Deserialization

--> Converts Python objects to JSON (serialization)
--> Converts JSON to Python objects (deserialization)
--> Useful for APIs and data exchange

Key Advantages:

--> Type-safe data handling
--> Cleaner and more readable code
--> Automatic validation
--> Reduces bugs in data pipelines and APIs
